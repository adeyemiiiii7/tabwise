# Changelog

All notable changes to Tabwise are documented here.

## [1.0.0] — 2025-05-20

### Added
- Automatic tab categorisation using OpenAI, Claude, or Gemini
- Smart offline categorisation — works without any API key using built-in pattern matching (~250 known domains + URL/title heuristics)
- Learns from corrections — reassign a tab once and Tabwise remembers that domain permanently
- Toggle AI on/off without losing your saved API key
- Daily screen time tracking per domain with live updates
- Dashboard: weekly screen time chart, category pie chart, top sites table
- Resources view: estimated RAM per open tab, idle tab detection, close-all-inactive action
- Tab groups: move any group to a new window in one click
- Onboarding flow with optional API key setup
- Settings: custom categories with emoji + colour, inactivity alerts, data export (CSV), full data reset
- Popup mode badge: AI mode / Auto mode / Setup needed with plain-English description
- Support for Chrome, Brave, and Dia
